# TrustEdge

TrustEdge — The Human Banking Platform (Predictive Customer Retention & AI-Powered RM Dashboard)

## Problem Statement

This project addresses the challenge of identifying and preventing customer churn proactively. TrustEdge monitors customer behavior (like transaction velocity, fee sensitivity, and complaint frequency) and uses machine learning to detect churn risk in real-time, empowering Relationship Managers (RMs) with actionable insights.

## How to Run Locally

1. Clone the repo: `git clone [TBD]`
2. Install dependencies for the server:
   ```bash
   cd server
   npm install
   ```
3. Setup the local database:
   ```bash
   npm run db:generate
   npm run db:migrate
   npm run db:seed
   ```
4. Start the server:
   ```bash
   npm run dev
   ```
5. In a new terminal, setup the client:
   ```bash
   cd client
   npm install
   npm run dev
   ```
6. Open browser at: `http://localhost:5173`

## Project Structure

/client — React frontend (Vite) and customer/RM dashboard UI

/docs — Architectural plans and project documentation

## Dataset

All data is 100% synthetic, generated by our team for testing. It simulates customer banking behavior including:
• App login frequency and digital transaction ratio
• Missed EMIs and cancelled FDs
• Complaint frequency and escalation flags

No real bank data was used.
